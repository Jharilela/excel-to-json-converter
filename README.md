## üöÄ Getting Started

### Prerequisites

- Node.js (version 16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd excel-to-json-converter
```

2. Install dependencies:
```bash
npm install
```

3. Set up Google Analytics (optional):
   - Copy `.env.example` to `.env`
   - Replace `G-XXXXXXXXXX` with your actual Google Analytics Measurement ID
   - Get your Measurement ID from [Google Analytics](https://analytics.google.com/)

4. Start the development server:
```bash
npm run dev
```

5. Open your browser and navigate to `http://localhost:5173`

### Google Analytics Setup

This application includes Google Analytics tracking for the following events:
- **Page Views**: When users visit the application
- **File Uploads**: File type and size when files are uploaded
- **File Conversions**: Number of files and output format when converting
- **Downloads**: Output format and file count when downloading
- **Settings Changes**: When users modify conversion settings
- **Errors**: Error types when issues occur
- **User Actions**: Clear all actions

To set up Google Analytics:

1. Create a Google Analytics 4 property
2. Get your Measurement ID (format: G-XXXXXXXXXX)
3. Copy `.env.example` to `.env`
4. Replace the placeholder with your actual Measurement ID:
   ```
   VITE_GOOGLE_ANALYTICS_ID=G-XXXXXXXXXX
   ```

The `.env` file is excluded from version control for security, but `.env.example` is tracked as a template.

### Building for Production

To create a production build:

```bash
npm run build
```

The built files will be in the `dist` directory.
